# ToraFinance

ToraFinance is an application for managing donations and scholarships for kollelim and yeshivot.

## Initial Product Scope

- Donor management and donation history
- Scholarship recipient records
- Monthly scholarship cycles
- Donation allocation tracking
- Receipts, reports, and audit-friendly exports
- Role-based access for administrators and staff

## Status

Project setup is starting. Architecture, stack, and first implementation milestones are still being defined.
